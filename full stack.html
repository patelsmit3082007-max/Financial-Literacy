For this single‑file demo, the “backend” is implemented with pure JavaScript:

  • Generates a synthetic price series with noise & drift
  • Computes a moving average + momentum signal
  • Projects a short‑term target price
  • Updates the UI and chart in real time

To turn this into a real app, expose an HTTP endpoint (/predict) from your backend (Python, Node, etc.) and call it from this page using fetch().
